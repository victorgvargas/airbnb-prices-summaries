# App Icons

To build properly, you should add these icon files:

- `icon.icns` - macOS icon (512x512 or larger)
- `icon.ico` - Windows icon (256x256 or larger) 
- `icon.png` - Linux icon (512x512 PNG)

You can create a simple icon or use an online converter to generate these from a single PNG image.

For now, the build will work without icons, but the apps will use default Electron icons.
